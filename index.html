FAQ
Häufige Fragen
Was genau ist TermNotch?
Eine native macOS-App, die die MacBook-Notch in ein vollwertiges Drop-Down Terminal verwandelt. Per Hotkey gleitet ein Terminal-Fenster aus der Notch — perfekt für schnelle Befehle und AI CLI Tools.
Welche AI Tools werden unterstützt?
Alle. TermNotch ist ein vollwertiger Terminal-Emulator. Claude Code, Gemini CLI, Codex CLI, GitHub Copilot CLI — alles was in der Kommandozeile läuft, läuft in TermNotch.
Brauche ich ein MacBook mit Notch?
TermNotch ist für MacBooks mit Notch optimiert (ab MacBook Pro 2021). Auf Macs ohne Notch erscheint das Terminal mittig am oberen Bildschirmrand.
Ist das ein Abo?
Nein. €9,99 einmalig. Alle Updates inklusive. Keine versteckten Kosten, kein Renewal.
Ist die App sicher?
Ja. Von Apple signiert und notarisiert. Keine Telemetrie, keine Datensammlung, keine Internetverbindung nötig. Alles läuft lokal.
Kann ich den Hotkey ändern?
Ja. Modifier (Ctrl, Opt, Cmd) und Taste (I, T, N, J, Space, `) frei kombinierbar. Standard: Ctrl+I.
Was ist die Local API?
Ein lokaler HTTP-Server auf Port 7777. Externe Tools können damit das Terminal steuern: Fenster öffnen, Befehle senden, Tabs erstellen, Benachrichtigungen anzeigen.
