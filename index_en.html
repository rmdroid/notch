FAQ
Frequently Asked Questions
What exactly is TermNotch?
A native macOS app that turns the MacBook notch into a full-featured drop-down terminal. Press a hotkey and a terminal slides out of the notch — perfect for quick commands and AI CLI tools.
Which AI tools are supported?
All of them. TermNotch is a full terminal emulator. Claude Code, Gemini CLI, Codex CLI, GitHub Copilot CLI — anything that runs in the command line runs in TermNotch.
Do I need a MacBook with a notch?
TermNotch is optimized for MacBooks with a notch (MacBook Pro 2021 and later). On Macs without a notch, the terminal appears centered at the top of the screen.
Is this a subscription?
No. €9.99 once. All updates included. No hidden costs, no renewal.
Is the app safe?
Yes. Signed and notarized by Apple. No telemetry, no data collection, no internet connection required. Everything runs locally.
Can I change the hotkey?
Yes. Modifier (Ctrl, Opt, Cmd) and key (I, T, N, J, Space, `) are freely combinable. Default: Ctrl+I.
What is the Local API?
A local HTTP server on port 7777. External tools can use it to control the terminal: open the window, send commands, create tabs, display notifications.
